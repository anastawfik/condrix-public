## Features

<CardGrid>
  <Card title="Multi-Machine Orchestration" icon="laptop">
    Run AI agent Cores on multiple machines — your desktop, a cloud server, a Raspberry Pi — and manage them all from a single interface. Maestro coordinates work across every Core.
  </Card>
  <Card title="Real-Time IDE Experience" icon="rocket">
    Full coding environment with Monaco editor, integrated terminal, Git operations, and live AI chat. Watch your agents write code, run tests, and commit changes as they happen.
  </Card>
  <Card title="Multiple Clients" icon="puzzle">
    Connect from anywhere: a web browser, a native desktop app (Tauri), a mobile app (React Native), or a terminal CLI (Ink). All clients are stateless and interchangeable.
  </Card>
  <Card title="Secure Remote Access" icon="approve-check-circle">
    Built-in Cloudflare Tunnel integration provides encrypted remote access to your Cores without exposing ports or configuring firewalls. One command to go from local to global.
  </Card>
  <Card title="Multi-Provider AI" icon="setting">
    Connect to Claude, OpenAI, Ollama, LM Studio, or any OpenAI-compatible endpoint. Create AI Profiles with automatic fallback chains — if one provider hits a rate limit, Condrix switches to the next without interruption.
  </Card>
  <Card title="Flexible Authentication" icon="random">
    Claude OAuth, API keys, local models with no auth, or custom endpoints. Each Core manages its own credentials securely, with registration tokens and token rotation for Maestro connections.
  </Card>
</CardGrid>

<Aside type="caution" title="Early Development">
  Condrix is under active development and not yet ready for production use. APIs, configuration formats, and features may change without notice. Contributions and feedback are welcome on [GitHub](https://github.com/anastawfik/condrix).
</Aside>
